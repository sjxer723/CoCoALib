IsPalindromic    --    test whether a univariate polynomial is palindromic


Syntax 
IsPalindromic(R: RINGELEM): BOOL

Description
This function tests whether its argument (univariate poly) is palindromic.

Example 
/**/  use P ::= QQ[x,y];
/**/  IsPalindromic(x+1);
true
/**/  IsPalindromic(y^4-3*y^2+1);
true
/**/  IsPalindromic(x-1);
false
